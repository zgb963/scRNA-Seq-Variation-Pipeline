This pipeline was built to re-map, cluster, and quantify scRNA-Seq data from Goodyer et al. to identify genes with high expression variance within mouse heart cell clusters.


Software Tools Required
=========

* Linux/Unix
* Python3
* Fastq dump
* Cell Ranger
* Seraut
* R
